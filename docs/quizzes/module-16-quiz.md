## Question 1

*Lesson L01 · Applications*


The intertemporal budget constraint is:


**Type:** Multiple Choice

- **A)** c1 + c2 = m1 + m2
- **B)** c1 + c2/(1+r) = m1 + m2/(1+r)
- **C)** c1/(1+r) + c2 = m1/(1+r) + m2
- **D)** c1 + c2 = (m1 + m2)/(1+r)

??? success "Show Answer"
    **Answer: B) c1 + c2/(1+r) = m1 + m2/(1+r)**

    Future consumption is discounted by (1+r)
