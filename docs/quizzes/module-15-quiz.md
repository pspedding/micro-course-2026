## Question 1

*Lesson L01 · Welfare*


The Slutsky equation is:


**Type:** Multiple Choice

- **A)** ∂x1/∂p1 = ∂x1^h/∂p1 - x1(∂x1/∂m)
- **B)** ∂x1/∂p1 = ∂x1/∂m - x1^h(∂x1^h/∂p1)
- **C)** ∂x1/∂p1 = ∂x1^h/∂p1 + x1(∂x1/∂m)
- **D)** ∂x1/∂p1 = ∂x1/∂m + x1^h(∂x1^h/∂p1)

??? success "Show Answer"
    **Answer: A) ∂x1/∂p1 = ∂x1^h/∂p1 - x1(∂x1/∂m)**

    Slutsky: Total effect = Substitution effect (Hicksian) + Income effect
