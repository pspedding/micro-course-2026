## Learning Objective

Analyze isoquant shapes and returns to scale properties for different production technologies.

---

## Isoquant Shapes and Technologies

Isoquants vary by production technology:
1. **Perfect Substitutes:** Linear isoquants (constant MRTS). Example: \(y = ax_1 + bx_2\).
2. **Perfect Complements (Leontief):** L-shaped isoquants. Example: \(y = \min(ax_1, bx_2)\).
3. **Cobb-Douglas:** Convex isoquants. Example: \(y = x_1^a x_2^b\).

### Returns to Scale:
- **CRS:** \(f(tx_1, tx_2) = tf(x_1, x_2)\) (e.g., Cobb-Douglas with \(a + b = 1\)).
- **IRS:** \(f(tx_1, tx_2) > tf(x_1, x_2)\) (e.g., \(a + b > 1\)).
- **DRS:** \(f(tx_1, tx_2) < tf(x_1, x_2)\) (e.g., \(a + b < 1\)).

**Euler’s Theorem:** For homogeneous functions, \(x_1 \frac{\partial f}{\partial x_1} + x_2 \frac{\partial f}{\partial x_2} = k f(x_1, x_2)\), where \(k\) is the degree of homogeneity.

---

## Worked Example

**[Australian context]**

An Australian solar panel manufacturer uses \(y = x_1^{0.6} x_2^{0.6}\):
1. **Returns to Scale:**
   \[
   f(tx_1, tx_2) = (tx_1)^{0.6} (tx_2)^{0.6} = t^{1.2} x_1^{0.6} x_2^{0.6} = t^{1.2} f(x_1, x_2) \quad \text{(IRS)}
   \]
2. **Euler’s Theorem:**
   \[
   x_1 (0.6 x_1^{-0.4} x_2^{0.6}) + x_2 (0.6 x_1^{0.6} x_2^{-0.4}) = 1.2 f(x_1, x_2)
   \]

---

## Common Misconception

> **"All Cobb-Douglas functions exhibit CRS."**

Only if exponents sum to 1. Here, \(0.6 + 0.6 = 1.2\) implies IRS.

---

## Key Takeaways

- Isoquant shape reveals input substitutability (linear → perfect substitutes).
- Returns to scale depend on the sum of exponents in Cobb-Douglas.
- Euler’s theorem links marginal products to total output for homogeneous functions.
