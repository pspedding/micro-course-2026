## Learning Objective

## Formal Monopoly Analysis

Consider a monopolist facing inverse demand function P(y) = a - by. The firm's cost function is C(y) = cy + F. Profit maximization requires MR = MC:

π(y) = P(y)y - C(y) = (a - by)y - cy - F

dπ/dy = a - 2by - c = 0 → y* = (a - c)/2b

Australian example: Suppose a local electricity provider faces P(y) = 100 - 0.5y with C(y) = 20y + 500. Then:

y* = (100 - 20)/1 = 80 units
P* = 100 - 0.5(80) = $60

Lerner Index: L = (60 - 20)/60 = 0.67

## Worked Example

**[Australian context]**

Sydney Water monopoly faces P(y) = 120 - y with MC = $30. Calculate:
1. Profit-maximizing output
2. Deadweight loss

Solution:
1. MR = 120 - 2y = 30 → y_m = 45
   P_m = 120 - 45 = $75
2. Competitive output where P=MC: 120 - y = 30 → y_c = 90
   DWL = 0.5×(75-30)×(90-45) = $1,012.50

---

## Common Misconception

> "Monopolies always produce less than competitive markets"

While typically true, this ignores durable goods monopolies (Coase conjecture) where monopoly may produce competitively to prevent future competition.

---

## Key Takeaways

- MR = P + y(dP/dy) reflects quantity effect on revenue
- Lerner index L = (P-MC)/P measures market power
- DWL exists because P > MC reduces total surplus
- Durable goods monopolies face time inconsistency problems
